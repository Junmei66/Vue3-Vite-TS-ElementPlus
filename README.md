# Vue3-Vite-TS-ElementPlus
Vite构建Vue3+TS+ElementPlus+Pinia+Echarts简易模板，使用Json-server模拟数据，包含用户登录，路由权限，Echart视图化，数据增删改查等基本功能


### 安装json-server
1. yarn add json-server
2. 在根目录创建db.json文件
3. json-server --watch db.json

## 项目安装启动
1. 安装：yarn
2. 运行：yarn dev
3. 运行mock: yarn mock
4. 启动：http://localhost:3000
